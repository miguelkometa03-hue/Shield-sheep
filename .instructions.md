## 🔑 Environment Variables Required

### Firebase Integration
```env
FIREBASE_API_KEY=your_firebase_api_key
FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
FIREBASE_PROJECT_ID=shield-sheep-xxxxx
FIREBASE_STORAGE_BUCKET=shield-sheep-xxxxx.appspot.com
FIREBASE_MESSAGING_SENDER_ID=your_sender_id
FIREBASE_APP_ID=your_app_id
```

### Cloudinary (Media Management)
```env
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_UPLOAD_PRESET=your_preset
CLOUDINARY_API_KEY=your_api_key
```

### AI Services
```env
# Jules.ia - Code Analysis & AI Suggestions
JULES_API_KEY=your_julesai_api_key
JULES_API_URL=https://api.julius.ia
JULES_MODEL=gpt-4-turbo

# Qwen Code - Repository Intelligence
QWEN_API_KEY=your_qwen_api_key
QWEN_MODEL=qwen-max
QWEN_REPOSITORY=miguelkometa03-hue/Shield-sheep
QWEN_BRANCH=main
```

## 🚀 Quick Start Commands

### Initialize Jules.ia
```bash
# 1. Install CLI
npm install -g @julesai/cli

# 2. Authenticate
jules login --api-key $JULES_API_KEY

# 3. Full analysis
jules analyze ./ --report=detailed
```

### Initialize Qwen Code
```bash
# Configure repository
qwen --connect miguelkometa03-hue/Shield-sheep --branch main

# Search code
qwen search "function name" --context=5

# Analyze performance
qwen analyze --focus=performance
```

### Verify Configuration
```bash
# Test Firebase connection
node -e "console.log(process.env.FIREBASE_PROJECT_ID)"

# Test Cloudinary setup
curl -X GET https://api.cloudinary.com/v1_1/${CLOUDINARY_CLOUD_NAME}/resources

# Test Jules.ia
curl -H "Authorization: Bearer $JULES_API_KEY" https://api.julius.ia/status
```
