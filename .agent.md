## Troubleshooting

### Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| `JULES_API_KEY not found` | Add to `.env` and reload |
| `Qwen cannot access repo` | Verify GitHub credentials |
| `Cloudinary 401 error` | Check API key in `.env` |
| `Firebase init failed` | Verify project ID matches |
| `CSP blocking scripts` | Update CSP in HTML headers |
